The application was created using the Django framework. Allows you to create tasks, mark performers and the status of completion. To work, you need to register and log in.

Build With:
- Python
- Django
- Bootstrap 5
- PostgreSQL
- Poetry
- Gunicorn
- Rollbar
- Flake8
